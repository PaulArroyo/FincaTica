Descripción del proyecto:
Un sistema web de control y visualización de resultados de la finca Abra&Mary S.A con el 
fin de que se pueda mejorar la toma de decisiones administrativas y tener una plataforma
accesible y amigable para los administradores.

Cómo instalar el repositorio en el equipo para desarrollo:
Debemos iniciar la terminal de git bash en la carpeta donde queremos clonar el repositorio y 
usaremos el link que nos provee la misma página.

Ejemplo del comando a utilizar:

git clone https://Poldrey@bitbucket.org/poldrey/repo-proyecto-grupo2.git

Al ser un repositorio privado debemos validarnos como usuarios y se nos descargará enseguida.
